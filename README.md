# Meny-app

En enkel meny- och inköpslista-app byggd med **Vite** och **vanilla JS**.

## Kom igång

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Miljövariabler

```
VITE_SUPABASE_URL=...
VITE_SUPABASE_ANON_KEY=...
```

## PWA

Manifest och service worker finns i `public/`.

## Skript

- `npm run lint`
- `npm run format`
